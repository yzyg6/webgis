# 城市漫游热门城市侧边栏设计

## 目标

在城市漫游页面添加热门城市侧边栏，点击城市按钮可直达该城市市中心，布局与火星探索的 `MarsFlightPanel` 保持一致。

## 数据结构

```typescript
interface HotCity {
  name: string       // 城市中文名
  nameEn: string     // 英文名
  lon: number        // 经度
  lat: number        // 纬度
  region: string     // 所属区域
  emoji: string      // 国旗 emoji
}
```

## 组件结构

### 新建文件

- `src/city/components/CityPanel.vue` — 热门城市侧边栏

### 修改文件

- `src/city/CityRoamingView.vue` — 添加 CityPanel 到布局，改为 grid 布局

## CityPanel.vue

### Props & Emits

```typescript
defineProps<{
  selectedCity: string | null
}>()

defineEmits<{
  'select-city': [city: HotCity]
}>()
```

### 模板结构

- 可折叠侧边栏（默认展开）
- 按区域分组显示城市按钮
- 每个按钮显示 emoji + 城市名
- 选中状态高亮

### 样式

- 宽度 220px，折叠时 40px
- 区域标签小字灰色
- 城市按钮：flex 布局，圆角 6px，hover 和 active 状态

## CityRoamingView.vue 变更

### 新增状态

```typescript
const selectedCity = ref<string | null>(null)
```

### 新增方法

```typescript
const flyToCity = (city: HotCity) => {
  if (!viewer) return
  selectedCity.value = city.nameEn
  viewer.camera.flyTo({
    destination: Cesium.Cartesian3.fromDegrees(city.lon, city.lat, 2000),
    orientation: {
      heading: 0,
      pitch: Cesium.Math.toRadians(-45),
      roll: 0,
    },
    duration: 2,
  })
}
```

### 布局变更

`.city-main` 改为 `display: grid; grid-template-columns: auto 1fr;`，与火星探索一致。
