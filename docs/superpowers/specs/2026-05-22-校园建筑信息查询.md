# 校园建筑信息查询 — 设计规格说明

## 概述

在 Cesium 3D 地球可视化项目中添加校园建筑信息查询功能。用户可以单击建筑查看摘要气泡卡片，展开为侧边详情面板，并通过搜索框和列表浏览定位建筑。

## 目标

- 单击 3D 地球上的任意校园建筑，显示摘要气泡（名称、用途、楼层、容纳人数）
- 展开气泡到侧边面板，显示完整建筑详情（元数据 + GeoJSON 属性）
- 按名称搜索建筑，实时过滤
- 在列表中浏览所有建筑，点击飞到任意建筑
- 保留所有现有功能（悬停提示、双击编辑、图层管理、底图切换）

## 非目标

- 建筑元数据的真实数据库集成（先用模拟数据，后续再接数据库）
- 室内导航或楼层平面图
- 设施预约/排期（后续阶段）
- 数据可视化大屏（后续阶段）

## 架构设计

### 数据层

**文件：** `src/data/campus-buildings.ts`

- 定义 `BuildingMeta` 类型，包含字段：name、purpose、floors、capacity、college、contact、description
- 提供 `MOCK_BUILDING_METAS`：以建筑名为 key 的 `Map<string, BuildingMeta>`
- 导出 `getBuildingMeta(name: string): BuildingMeta | null` 用于查询
- 导出 `getAllBuildingMetas(): BuildingMeta[]` 用于列表渲染
- 预留 `loadBuildingMetaFromDb()` 异步函数签名，供后续数据库集成

```ts
type BuildingMeta = {
  name: string;
  purpose: string;
  floors: number;
  capacity: number;
  college: string;
  contact: string;
  description?: string;
};
```

### 组件

#### `BuildingPopup.vue` — 摘要气泡卡片

- **Props：** `visible`、`x`、`y`、`buildingName`、`purpose`、`floors`、`capacity`
- **Emits：** `show-detail`、`close`
- 绝对定位在点击位置附近
- 深色半透明背景，圆角，与现有 HoverTooltip 风格统一
- 底部"查看详情"按钮，点击 emit `show-detail`

#### `BuildingInfoPanel.vue` — 侧边详情面板

- **Props：** `buildings`（完整列表）、`selectedBuilding`、`searchQuery`
- **Emits：** `select-building`、`update:searchQuery`
- **区域划分：**
  1. 顶部搜索框，实时过滤
  2. 建筑列表（可滚动），每项可点击飞到建筑
  3. 详情区域：选中建筑后展示 BuildingMeta 字段 + GeoJSON 原始属性
- 位于侧边栏 `LayerPanel` 下方

### 状态管理（App.vue）

新增响应式状态：

```ts
const selectedBuilding = ref<BuildingMeta | null>(null);
const buildingPopupInfo = ref<{
  x: number; y: number; layerId: string; featureIndex: number;
  properties: Record<string, unknown>;
} | null>(null);
const searchQuery = ref<string>('');
```

### 交互流程

#### 流程 1：单击建筑 → 气泡 → 详情

1. 用户单击 Cesium 地球上的建筑实体
2. `CesiumViewer` emit `select-entity`，携带屏幕坐标、图层 ID、要素索引、属性
3. `App.vue` 设置 `buildingPopupInfo` → `BuildingPopup` 出现在点击位置
4. 用户点击气泡中的"查看详情"
5. `App.vue` 根据建筑名查找 `BuildingMeta` → 设置 `selectedBuilding`
6. `BuildingInfoPanel` 展示完整详情

#### 流程 2：搜索与导航

1. 用户在 `BuildingInfoPanel` 搜索框输入关键词
2. 建筑列表实时按名称过滤
3. 用户点击列表中的建筑
4. `App.vue` 通过 `defineExpose` 调用 `CesiumViewer.flyToBuildingByName(name)`
5. `CesiumViewer` 查找实体，计算包围球，飞到该位置并高亮

#### 流程 3：关闭气泡

- 点击空白处或按 Escape → `buildingPopupInfo` 设为 null → 气泡关闭

### CesiumViewer 改动

- **新 emit：** `select-entity`（单击，区别于现有的 `hover-entity` 悬停事件）
- **新方法：** `flyToEntity(entity)` — 使用实体多边形的 `BoundingSphere` 计算相机目标，带偏移高度飞到该位置
- **新方法：** `flyToBuildingByName(name: string)` — 在所有 cityModelDataSources 中搜索匹配名称的实体，调用 `flyToEntity`
- **通过 `defineExpose` 暴露：** `flyToBuildingByName`
- 在现有 hover 和双击处理器旁添加单击处理器

## 保留功能

- 悬停高亮和 HoverTooltip（鼠标移动）
- 双击编辑属性（EditPanel）
- 图层管理（添加/删除/切换可见性）
- 底图切换（OSM/ArcGIS/Carto）
- GeoJSON 规范化流程
- 数据库图层同步（PostgreSQL + Express API）
- localStorage 持久化

## 模拟数据

初始模拟数据覆盖 `fynu_xihu_campus.geojson` 数据集中的 5-10 栋建筑。每条记录包含名称、用途、楼层、容纳人数、所属学院、联系电话和可选描述。模拟数据模块设计为只需实现 `loadBuildingMetaFromDb()` 即可替换为数据库调用。
