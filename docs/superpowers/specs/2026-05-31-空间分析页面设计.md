# 空间分析页面设计

## 概述

新增第四个功能模块「空间分析」，路由 `/analysis`，首期实现可视域分析功能。布局与现有三个子页面（校园展示、火星探索、城市漫游）保持一致，采用侧边参数面板 + Cesium 3D 地图的经典结构。

不使用现有 `shared/cdk/` 中的空间分析模块，核心可视域分析代码由用户后续提供，本次仅完成页面布局、交互框架和预留接口。

## 设计决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 导航入口 | 左侧菜单新增第四项 | 与现有三个模块并列，结构清晰 |
| 页面布局 | 侧边参数面板（方案 A） | 与其他三个子页面风格一致 |
| 结果展示 | 仅地图渲染 | 可视/不可见区域直接在地球上着色，无需额外面板 |
| 核心分析代码 | 预留接口，用户后续提供 | 用户已有 `viewshed.js` 实现，直接集成 |
| 模块结构 | 独立 `analysis/` 目录 | 与 `mars/`、`city/` 平级，职责隔离 |

## 组件设计

### AnalysisView.vue

主视图组件，负责 Cesium.Viewer 生命周期管理和分析状态协调。

**核心职责**：
- `onMounted` 初始化 Cesium.Viewer
- 加载 `CesiumTerrainProvider` 地形
- 开启 `scene.globe.depthTestAgainstTerrain = true`
- 管理 `viewshed` 实例（用户后续提供的核心代码）
- 监听 `ScreenSpaceEventHandler` 地图点击事件，获取观察点坐标

### ViewshedPanel.vue

侧边参数面板，参照 `CityPanel.vue` 的可折叠设计。

**Props**：

```ts
interface ViewshedPanelProps {
  isPicking: boolean       // 是否正在选取观察点
  isAnalyzing: boolean     // 是否正在分析
  hasResult: boolean       // 是否有分析结果
  observerHeight: number   // 观察者高度
  targetHeight: number     // 目标高度
  radius: number           // 分析半径
}
```

**Emits**：

```ts
'update:observerHeight'   // 观察者高度变更
'update:targetHeight'     // 目标高度变更
'update:radius'           // 分析半径变更
'start-pick'              // 开始选取观察点
'start-analyze'           // 开始分析
'clear-result'            // 清除结果
```

**面板内容**：

1. **标题区**：「⊕ 可视域分析」+ 折叠按钮（`‹` / `›`）
2. **参数区**：
   - 观察者高度（qdOffset）：默认 2m，范围 0.5-50m，步长 0.5
   - 目标高度（zdOffset）：默认 2m，范围 0.5-100m，步长 0.5
   - 分析半径：默认 500m，范围 100-5000m，步长 50
3. **操作区**：
   - 「📍 选取观察点」按钮 — 发出 `start-pick` 事件，进入地图点选模式
   - 「▶ 开始分析」按钮 — 发出 `start-analyze` 事件（观察点已选取后可用）
   - 「✕ 清除结果」按钮 — 发出 `clear-result` 事件（有结果时可用）
4. **状态区**：底部显示当前状态文字
   - 等待选取：「请在地图上点击选取观察点」
   - 已选取：「观察点已设置，点击开始分析」
   - 分析中：「正在计算可视域...」
   - 完成：「分析完成」
   - 已清除：「结果已清除」

## 地形配置

`AnalysisView.vue` 的 `onMounted` 中加载地形：

```ts
viewer.terrainProvider = await Cesium.CesiumTerrainProvider.fromUrl(
  Cesium.IonResource.fromAssetId(1),  // Cesium World Terrain
  { requestWaterMask: true }
)
viewer.scene.globe.depthTestAgainstTerrain = true
```

> 注：地形源可替换为其他服务（如 mars3d 地形），视网络环境而定。

## 样式规范
